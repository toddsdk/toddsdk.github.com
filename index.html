byteSelf - iPad Game

byteSelf is a 3-player local strategy-puzzle game for the iPad. I have programmed all the aspects of this game, including graphics, animation, gameplay, SFX, music and UI. We used C# and Unity3D. It was released to the iPad App Store in 2012, but unfortunately is not available to download anymore.

ICS - Interactive Art Installation

ICS (Infundibulum Crono-Sinclástico) was an interactive art installation, which made use of technologies such as VR glasses and Microsoft Kinect for motion-control. I was responsible for implementing its 3D graphics, motion-based controls, UI and sound effects. We used C# with Unity3D and add-ons.
